프로젝트 유의 사항

  • 프로젝트를 수강생이 직접 진행하지 않고 수강생 이외의 인원이 코드를 대리 작성하는 행위는 엄격히 금지됨.(발견 즉시 F)
  • 모든 팀원이 프로젝트 코드 개발에 참여야하여야 하며 Git Commit Log에 1회 이상 등장하지 않을 경우 프로젝트 미참여로 간주.
  • 모든 코드는 Python 3.7.x / Django 2.2.x 에서 정상동작 하도록 작성되어야함.
  • 강의 자료 이외의 Django 관련 서적, 인터넷 자료 검색 등을 적극적으로 활용할 것이 권장 됨.
  • Django Project는 PyCharm을 사용하지 않을 경우 개발에 많은 어려움이 따르므로, 반드시 PyCharm 및 Debugging 기법을 숙지 할 것
