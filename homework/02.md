## Example of Turtle Graphics 01(Basic Star Example)
```
import turtle
 
t = turtle.Turtle()
 
for i in range(5):
    t.forward(150)
    t.right(144)
turtle.done()
```
- Output :
![Turtle Star](../assets/img/hw/02/01.jpg)

## Example of Turtle Graphics 02(Spiral Star Example)
```
import turtle 

spiral = turtle.Turtle()

for i in range(20):
    spiral.forward(i * 10)
    spiral.right(144)
    
turtle.done()
```
- Output :
![Turtle Star](../assets/img/hw/02/02.png)
